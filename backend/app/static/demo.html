SEDUC Simulados — Demonstração de Integração

Sistema de Simulados Educacionais com IA · Secretaria de Educação de Sergipe

Frontend (esta página) consumindo a API FastAPI ao vivo · banco de dados real

1. Gerar prova

O backend filtra o banco, equilibra por dificuldade e embaralha as alternativas.

2. Cadastrar questão

Formulário envia para a API, que valida e grava no banco.

3. Banco de questões

Lista lida diretamente do banco via GET /questoes.

4. Ciclo de simulado — do gestor ao aluno

Fluxo completo: o gestor cria e gera, libera para a turma, o aluno responde e o sistema corrige automaticamente.
